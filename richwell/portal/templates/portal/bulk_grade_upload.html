How to Upload Grades

  1. Download the template CSV file or prepare your own following the format below
  2. Fill in the student IDs and corresponding grades
  3. Upload the completed CSV file using the form below
  4. Review and confirm the grades before final submission

CSV Format

Your CSV file should have the following format:

student_id,grade
2021-00001,1.75
2021-00002,2.00
2021-00003,1.50

Valid Grades:

  • 1.00 - 3.00: Passing grades
  • 4.00: Conditional
  • 5.00: Failed
  • INC: Incomplete
  • DRP: Dropped
  • P: Passed

Requirements:

  • File must be in CSV format (.csv)
  • Maximum file size: 5MB
  • First row must be headers
  • Student IDs must be valid
  • Grades must match valid values
